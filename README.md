# **TrackIt**

🎯 **TrackIt** — минималистичное и удобное приложение для ведения счёта в настольных играх. Поддерживает несколько режимов игры, управление игроками, историю партий, настраиваемые темы и работу офлайн.

---

## 🆕 Что нового в версии 3.4

- **Новый режим 🎯 Цель** — победитель тот, кто первым набирает нужное количество очков. Теперь доступны три режима: Обнуление, Проигрыш и Цель
- **Обновлена главная страница** — появилась живая статистика: количество комнат, игроков, сыгранных партий и лидерборд побед. Обновляется без перезагрузки
- **Обновлено добавление очков** — кнопка `+` / `−` позволяет добавлять или снимать очки. В истории ходов появилась кнопка `✕` для отмены записи
- **Обновлена страница Истории** — карточки партий теперь показывают режим игры, итоговые места всех участников, медали, и дату. Добавлен поиск по названию комнаты
- **Обновлены темы** — добавлен glass-эффект, обновлена стилистика карточек и модальных окон
- **Исправлен баг двойного добавления очков** — повторное нажатие кнопки подтверждения больше не дублирует запись
- **Исправлен баг зависшего листенера** — модальное окно добавления игрока теперь корректно закрывается в любой ситуации

---

## 🚀 Функции

- **Комнаты** — создавай отдельные комнаты для разных игр и компаний
- **Три режима игры** — Обнуление (перебор сбрасывает счёт), Проигрыш (перебор = выбывание), Цель (победа при достижении порога)
- **Управление игроками** — аватарки, история ходов, личная статистика и титулы
- **История партий** — архив всех завершённых игр с поиском, счётом и режимом
- **Статистика на главной** — лидерборд, чемпион по победам, антирекорд
- **Темы оформления** — 15+ тем, автоопределение системной тёмной/светлой темы
- **Офлайн** — полностью работает без интернета (PWA, Service Worker)
- **Локальное хранение** — все данные в браузере, ничего не теряется
